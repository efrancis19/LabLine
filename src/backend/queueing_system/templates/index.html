LabLine

LabLine is a system that allows for more efficient lab operations by streamlining the process of students requesting assistance from a lab tutor.

Features

LabLine's features include a request page where students can provide their PC number and a description of the problem they're encountering. They can also cancel their request if they no longer need help and view their previous requests.

Instructions

To get started, register as a new user by clicking "Register". Then, enter your username, email address, and password, and also indicate if you are registering as a student or as a tutor.
