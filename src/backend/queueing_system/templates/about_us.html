About LabLine

Mission Statement

Driven by accessibility, we strive to enhance the learning environment in universities by providing systems to improve accessibility to help for students.

Our Target Market

The target market for our system is aimed towards universities with busy computer labs. We aim to focus initially on the Irish market and later expand further abroad.

Our Product

LabLine is a system for universities that strives to provide more accessible assistance to students and a less stressful experience for lab tutors.

System Features

LabLine includes the following features as part of the system:

  • Authentication system for both students and lab tutors to access the system.
  • Help request system for students who can request assistance from a lab tutor. The student inputs their PC number and a description of their request which will be received by a lab tutor when they click send.
  • View Requests for each tutor to view a list of all the requests assigned to them.
  • Cancel request option for students to cancel a request if it has been already resolved.
  • Accept request option for tutors to accept a pending request on their dashboard.
  • Mark request as completed option for tutors to indicate a request is complete once it has been resolved.

Technologies Used

  • HTML, CSS and JavaScript for the frontend.
  • Django and Python for the backend.
  • Channels and WebSocket to handle request messaging from students to tutors.
